Mulai Perjalanan Codingmu

tentang program rpl

Jurusan Rekayasa Perangkat Lunak (RPL) memfokuskan pada pengembangan kemampuan merancang, membuat, dan memelihara aplikasi berbasis komputer sesuai kebutuhan industri. Di sini, siswa akan mempelajari alur lengkap pembuatan perangkat lunak — mulai dari analisis kebutuhan, perancangan antarmuka, pemrograman, hingga pengujian dan dokumentasi — dengan pendekatan praktik langsung yang menyiapkan mereka menjadi developer profesional. Dengan kurikulum terkini dan proyek kolaboratif, jurusan RPL menanamkan kreativitas, logika, dan keterampilan problem-solving, sehingga lulusannya siap bersaing di era digital yang terus berkembang

Apa yang Dipelajari di RPL?

Pemrograman

Belajar membuat aplikasi dan website menggunakan bahasa populer seperti Java, Python dan PHP, serta teknologi terkini (Laravel, React)

Database

Merancang dan mengelola sistem penyimpanan data untuk aplikasi/website dengan tools profesional (MySQL, PostgreSQL)

DevOps

Mempelajari cara mengembangkan dan mengoperasikan aplikasi secara efisien dengan tools modern (Docker, Jenkins)

Fasilitas Di Jurusan RPL

  • Lab Komputer High-End

    PC dengan processor i7-13700k, RAM 16 GB, SSD 512GB dan VGA RTX 3060 Ti. untuk menjalakan aplikasi pemeograman & desain lancar

  • Tools Profesional

    Software terkini seperti Visual Studio Code, Git, Docker, dan lisensi resmi JetBrains (IntelliJ, PyCharm)

  • Internet Cepat & Stabil

    Koneksi 100 Mbps untuk pembelajaran cloud computing, streaming materi dan kolaborasi online

  • Sertifikat Industri

    Akses ke pelatihan dan ujian sertifikat Microsoft, Cisco dan Oracle untuk siswa/i
