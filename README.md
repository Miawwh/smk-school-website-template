## Installation

1. Clone this repository:  
   ```bash
   git clone https://github.com/Miawwh/smk-school-website-template.git
   ```  
2. Open `index.html` in your browser.  

> **Note:** No additional dependencies are required; pure HTML, CSS, and Bootstrap 5 are used.  

---

## Usage

- Customize the school name by editing the `<title>` and relevant headings in `index.html`.  
- Add new pages by copying `index.html`, renaming the file, and updating the navigation links accordingly.  

---

## Features

- **Responsive design** using Bootstrap 5 grid and components.  
- **Customizable**—replace placeholder text and images with your own.  
- **Clean structure** with separate folders for CSS, JS (if needed), and assets.
